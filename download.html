SkillSwap Source Code

This is the source code for the SkillSwap platform, a skill exchange application where users can share skills they can teach and find others with skills they want to learn. It features skill testing and rating functionality to ensure quality exchanges.

Download Source Code (ZIP)

How to use this code:

  1. Download the ZIP file using the button above
  2. Extract the ZIP file to your local machine
  3. Upload it to your GitHub repository
  4. Install dependencies with npm install
  5. Create a PostgreSQL database and update the connection string in your environment variables
  6. Run the database migrations with npm run db:push
  7. Start the development server with npm run dev
